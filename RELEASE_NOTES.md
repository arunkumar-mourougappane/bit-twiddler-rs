## What's new

- Initial release of Bit Twiddler — a cross-platform developer toolbox built with Tauri v2.

## Installation

Download the installer for your platform from the assets below, or install via Cargo:

```sh
cargo install bit-twiddler
```

## System requirements

| Platform | Requirement |
|---|---|
| macOS | macOS 11+ (WebKit built-in) |
| Linux | `libwebkit2gtk-4.1` — install with `sudo apt install libwebkit2gtk-4.1-dev` |
| Windows | WebView2 (ships with Windows 10/11) |
